Lung Cancer Detection

Summary

Machine Learning project which uses X-ray scans to classify and detect lung cancer nodules.

Dataset was provided through Kaggle and was preprocessed, cleaned and passed through a Neural Network using Tensorflow. Pre trained model architectures were evaluated for the task, VGG16, AlexNet and ResNet50 models were implemented.

Models were evaluated with the VGG-16 architecture achieving a validation accuracy of 87% and testing accuracy of 73%. Metrics and hypertuning techniques were used for evaluating and increasing model's performance.

Bounding boxes were implemented to identify location of nodules. Final report discusses further findings.

NFL Combine Prediction

Summary

Machine Learning project which uses official NFL combine data to classify if a player will achieve Pro Bowl level.

Combine data was webscraped using selenium framework dating back from 1987. Data was preprocessed, cleaned and passed through a Neural Network using Tensorflow. Final model achieved validation accuracy of 65.9% and testing accuracy of 69.6%

Implemented feature engineering and hypertuning techniques in order to increase accuracy. Determined metrics for evaluating the model's performance and created further visualizations.

Website was created using Django3, and final model was deployed on Heroku services. Further visualizations comparing the 2023 combine class were created and can be seen on website.

To Do list Website

Summary

ToDoneNow is a user-friendly and efficient to-do list website built on the Django3 framework and deployed through Heroku. It offers a streamlined task management experience, helping users stay organized and productive with ease.

ToDoneNow allows for Task Creation, Editing and Prioritization where users can add tasks, mark them as important, edit them and mark them as complete when finished.

ToDoneNow also has User Accounts and Password Authentication which allows access to their to-do lists from any device and is password protected. It also is Cloud deployed on Heroku which provides Data Synchronization in real-time, ensuring that your tasks are always up-to-date, even on different platforms.

ToDoneNow is the ultimate solution for individuals seeking an efficient, reliable, and user-friendly to-do list application.
